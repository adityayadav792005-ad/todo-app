# Todo Application

A full-stack Todo Application built using React, Spring Boot, and MySQL.

## Features

- User Registration
- User Login
- Add Todo
- View Todo List
- Delete Todo
- REST API using Spring Boot
- MySQL Database Integration
- React Frontend

## Tech Stack

### Frontend

- React
- Vite
- Axios
- CSS

### Backend

- Spring Boot
- Spring Data JPA
- Hibernate

### Database

- MySQL

## Project Structure

todo-app

├── todo-frontend (React Frontend)

└── todo (Spring Boot Backend)

## API Features

### User APIs

- Register User
- Login User

### Todo APIs

- Add Todo
- Get Todos
- Delete Todo

## How to Run

### Backend

1. Open backend project
2. Configure MySQL in application.properties
3. Run Spring Boot Application

### Frontend

1. Open frontend folder
2. Install dependencies

npm install

3. Start application

npm run dev
